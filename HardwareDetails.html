Hardware Details Tool

Key Features

  • No Admin Rights Required: Safe operation without elevated privileges
  • Quick Hardware Summary: Instant access to Model, BIOS, Serial Number & Hostname
  • Easy Driver Lookup: Comprehensive driver info for various hardware components
  • Offline Operation: Runs completely locally without internet dependency

How to Use

  1. Download the zip file
  2. Extract and run the command file
  3. The tool will automatically display: Make / Model, BIOS Version, Serial Number, Hostname
  4. After that, it will ask:
    Do you want driver details? (y/n):
  5. Type y and press Enter to view driver details.
  6. Select the driver number you want to check:
1. Audio
2. Battery
3. Biometric
4. Bluetooth
5. Camera
6. Display
7. Keyboard
8. Network
9. Touchpad
10. USB

For example, type 5 and press Enter to check the Camera driver. It will again ask if you want to check another driver. Type y to continue or n to exit.
